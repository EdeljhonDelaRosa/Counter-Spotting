────────────────────────────────────────────
        COUNTER-SPOTTING QUICK START
────────────────────────────────────────────

1. Install & Enable
─────────────────
✔ Load the extension via chrome://extensions/
✔ Enable Developer Mode → Load unpacked
✔ Ensure icon appears in toolbar

2. Open Popup Panel
─────────────────
Click the Counter-Spotting icon near the address bar

3. Configure Settings
─────────────────
[ ] Enable Proxy Routing       → anonymizes traffic
[ ] Enable Decoy Traffic       → generates harmless noise
Decoy Rate: 0.25               → default, adjust 0–1

4. Start Browsing Social Media
──────────────────────────────
• Platforms: Facebook, Twitter/X, Instagram, TikTok, YouTube
• Interact normally: Like, comment, share
• Counter-Spotting automatically:
    - Sanitizes URLs
    - Blocks trackers
    - Routes traffic through proxies (if enabled)
    - Sends decoy requests in the background

5. Recommended Tips
─────────────────
• Use with privacy-focused browsers (Chrome, Brave, Edge)
• Regularly update extension to include latest tracker rules
